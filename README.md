# Auth App

A simple authentication and user management backend built with Spring Boot.  
It supports user registration with basic validations and demonstrates clean service-layer architecture.

---

## 🚀 Features
- User registration
- Email validation (required & unique check)
- DTO-based request/response handling
- Service layer with business logic separation
- Unit testing with JUnit & Mockito

---

## 🛠 Tech Stack
- Java
- Spring Boot
- Maven
- MySQL

---

## 📦 Key Dependencies
- Spring Web
- Spring Security
- Spring Data JPA
- MySQL Driver
- ModelMapper
- JUnit 5
- Mockito
